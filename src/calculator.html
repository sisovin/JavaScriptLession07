Lesson 07: Functions - 'updateCalculation' and reuse the code.

This exercise 7j, 7k, and 7l are about creating a calculator project. The calculator project is coming from the exercise 5r (Calculator project with duplication code). The exercise 7j is about creating a function to update the calculation. The exercise 7k is about creating a function to update the display. The exercise 7l is about reusing the code from the exercise 7j and 7k to create a calculator project. The challenges are to update the existing duplicate code to the function as the below guidances:

  • Create a function called 'updateCalculation' that takes a value and appends it to the calculation variable.
    • Notice there's a lot of duplicated code in the buttons. You can use the 'updateCalculation' function to reduce the duplication.
    • Create 'updateCalculation' function and reuse the code and append it to the calculation variable.
  • Create a copy of the Cart Quantity project from exercise 6l.
    • Create an 'updateCartQuantity' function that takes a value and appends it to the cart quantity variable.
  • Modify the 'updateCartQuantity' function from 7h so that if the quantity is invalid, alert(); and then return; (this is called an early return). An early return make our code cleaner because we can remove the 'else-if' / 'else' block.
