Lesson 07: Functions | Multiple Parameter with Temperature Conversion in JavaScript

This exercise 7d create 'convertToFahrenheit' function that takes a 'celsius' parameter and returns a number in degrees Fahrenheit. Then, conditions are checked and the result is displayed in the console as the followings:

  • The formula to convert Celsius to Fahrenheit is: (celsius * 9/5) + 32.
  • Create a function called 'convertToFahrenheit(25)'=> 77


With the exercise 7e, create 'convertToCelsius' function that takes a 'fahrenheit' parameter and returns a number in degrees Celsius. Then, conditions are checked and the result is displayed in the console as the followings:

  • The formula to convert Fahrenheit to Celsius is: (fahrenheit - 32) * 5/9.
  • Create a function called 'convertToCelsius(86)'=> 30


With the exercise 7f, create 'convertTemperature' function that takes a 'degrees' and 'unit' parameter with a number and a unit('C' for Celsius, 'F' for Fahrenheit) and returns a number in degrees Fahrenheit or Celsius. Then, conditions are checked and the result is displayed in the console as the followings:

  • Create a function called 'convertTemperature(25, 'C')'=> '77F'
  • Create a function called 'convertTemperature(86, 'F')'=> '30C'
