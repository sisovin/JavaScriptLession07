Lesson 07: Function on Length Conversion in JavaScript

This exercise 7g create 'convertLength' function that takes a 'length', 'from' and 'to' parameter with a number and a unit('km' for kilometers, or 'miles') and convert the length to kilometers or miles. Note: 1 mile is equal to 1.6 km (approximately). Then, conditions are checked and the result is displayed in the console as the followings:

  • convertLength(50, 'km', 'miles') => '80 km'
  • convertLength(32, 'miles', 'km') => '20 miles'
  • convertLength(50, 'km', 'km') => '50 km'


With the exercise 7h, update the 'convertLength' function to support converting between km, miles, and also feet. Note: 1 mile is equal to 1.6 km (approximately) and 1 km is equal to 3281 ft (approximately). Then, conditions are checked and the result is displayed in the console as the followings:

  • convertLength(5, 'miles', 'km') => '8 km'
  • convertLength(5, 'miles', 'ft') => '26400 ft'
  • convertLength(5, 'km', 'ft') => '16405 ft'


With the exercise 7i, update the 'convertLength' function so that if you give an invalid unit, it will return `Invalid unit: ${unit}`. Then, conditions are checked and the result is displayed in the console as the followings:

  • convertLength(5, 'lbs', 'lbs') => 'Invalid unit: lbs'
