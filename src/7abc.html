Lesson 07: Functions | Simple Parameter in JavaScript

In this Exercise, we create a 'greet' function that display the messages 'Hello!' in the console. Then, we call and run this function a few times using greet(). We also create a 'greetName' function that takes a name parameter and display the message `Hello, ${name}!` in the console. We call and run this function a few times using greetName('Sisovin'). Lastly, we try calling the greetName function without passing the parameter 'name' (it will display 'Hello, undefined!') and Modify the function so that if 'name' undefined (it will display 'Hello, there!') instead in the console. Hint: use an if-Statement. Since undefined is falsy value, you can use: if (!name) { ... } to check if the name is undefined).

  • 'greet' function that display the messages 'Hello!' in the console.
  • 'greetName' function that takes a 'name' parameter display the message `Hello, ${name}!` in the console.
  • 'greetName' function without passing the parameter 'name' (it will display 'Hello, undefined!') and Modify the function so that if 'name' undefined (it will display 'Hello, there!') instead in the console.
